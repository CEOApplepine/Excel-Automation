# Sales Reporting Automation using Excel VBA

## Overview
Automates monthly sales reporting using Excel VBA macros, reducing manual work and improving accuracy.

## Features
- Generates PivotTables and charts automatically
- Click a button to refresh reports
- User-friendly input forms for data entry

## Instructions
1. Open `MonthlyReport.xlsm` in Excel
2. Enable macros if prompted
3. Click the "Generate Report" button on the Dashboard
4. PivotTables and charts will refresh automatically
